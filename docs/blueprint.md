# **App Name**: ApnaBandhan

## Core Features:

- Wedding Invitation Video Showcase: Display a gallery of wedding invitation video samples, with preview player and individual pricing.
- Wedding Invitation Card Gallery: Display a grid-based gallery of wedding invitation cards with filtering options (color, B&W, digital, CDR).
- Service Detail Page: Display detailed information for each service, including samples, descriptions, inclusions, timelines, and pricing, with an 'Order Now' button.
- Order/Booking Form: Capture client details (name, phone, email, wedding date, selected service, notes) and save orders to the database and also send a WhatsApp message.
- Admin Panel: Secure admin login with a dashboard to view orders, manage client details, control order status, upload files, manage services and prices, manage combo packages, track payments, and export orders to Excel.
- Order management with payment: Handle payment through UPI, Google Pay, PhonePe, and Paytm. Update the database of order status from pending to delivered, and also manage payments (optional/advance).
- AI Powered Image Enhancer: Automatically enhance uploaded wedding photos and videos, acting as a tool to make professional touch-ups to existing media.

## Style Guidelines:

- Primary color: Soft gold (#D4AF37) to convey premium and traditional feel.
- Background color: Off-white (#FAF9F6) for a clean, modern aesthetic.
- Accent color: Maroon (#800000) for highlights and call-to-action buttons, complementing the gold and white.
- Font pairing: 'Poppins' (sans-serif) for headings and 'PT Sans' (sans-serif) for body text to ensure modern, readable typography.
- Use elegant, custom-designed icons related to wedding themes, with a soft gold fill and subtle line details.
- Mobile-first, responsive layout with clear sections, generous spacing, and intuitive navigation. Grid system for galleries and pricing tables.
- Subtle micro-animations like hover effects, smooth fade-ins, and button ripples to enhance interactivity without being distracting.
